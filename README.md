
## Description

Небольшой сервис для проверки ip и получения данных о том используются ли VPN, Прокси , Ralay , TOR . Собран на основе api vpnapi.io [vpnapi.io](https://vpnapi.io)


## Installation

```bash
$ npm install
$ mv .env-example .env

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
